## 高阶函数

高阶函数是指一个函数可以接收一个或多个函数作为参数，或者返回一个函数。

### 常见的高阶函数

1. 数组的一些内置方法：forEach、map、filter、reduce、some、every
2. 定时器：setTimeout、setInterval
3. 事件绑定：addEventListener
4. 异步任务： Promise
   ...

### 柯里化

```js
function add(a) {
  return function(b) {
    return function(c) {
      return a + b + c;
  }
}

const res = add(1)(2)(3)
console.log(res) //6
```
