## 判断数据类型的方法

1. **typeof** 适用于判断基础类型，对于引用类型，除了 function 会返回 function，其他都会返回 object

```js
typeof 1; //"number"
typeof "1"; //"string"
typeof true; //"boolean"
typeof undefined; //"undefined"
typeof null; //"object" 历史遗留问题，null 是一个空指针，指向空对象，因此 typeof null 会返回 object
typeof Symbol(); //"symbol"
typeof []; //"object"
typeof {}; //"object"
typeof function () {}; //"function"
```

2. **instanceof** 适用于判断引用类型，判断是否为某个构造函数的实例, 但是不能用于判断基础类型

```js
[] instanceof Array //true
[] instanceof Object //true
{} instanceof Object //true
{} instanceof Array //false
```

3. **constructor** 适用于判断引用类型，判断实例的构造函数, 但是不能用于判断 **null** 和 **undefined**, 因为 它们 没有构造函数

```js
let num = 1;
num.constructor === Number; //true

let str = "1";
str.constructor === String; //true

let bool = true;
bool.constructor === Boolean; //true

let symbol = Symbol();
symbol.constructor === Symbol; //true

let bigInt = BigInt(100);
bigInt.constructor === BigInt; //true

let arr = [];
arr.constructor === Array; //true

let obj = {};
obj.constructor === Object; //true

let fn = function () {};
fn.constructor === Function; //true

class Person {}
let person = new Person();
person.constructor === Person; //true
```

4. **Array.isArray()** 适用于判断数组

```js
Array.isArray([]); //true
Array.isArray({}); //false
```

5. **Object.prototype.toString.call()** 适用于判断所有数据类型

```js
let num = 1;
Object.prototype.toString.call(num).replace(/\[object\s(.+)\]/, "$1"); //"Number"

let str = "1";
Object.prototype.toString.call(str).replace(/\[object\s(.+)\]/, "$1"); //"String"

let bool = true;
Object.prototype.toString.call(bool).replace(/\[object\s(.+)\]/, "$1"); //"Boolean"

let fn = function () {};
Object.prototype.toString.call(fn).replace(/\[object\s(.+)\]/, "$1"); //"Function"

let obj = {};
Object.prototype.toString.call(obj).replace(/\[object\s(.+)\]/, "$1"); //"Object"

let arr = [];
Object.prototype.toString.call(arr).replace(/\[object\s(.+)\]/, "$1"); //"Array"

let date = new Date();
Object.prototype.toString.call(date).replace(/\[object\s(.+)\]/, "$1"); //"Date"
```
